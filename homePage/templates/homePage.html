A bit more about me...

Hello, I am a full stack software/web developer from Copenhagen, Denmark currently looking for work. My primary languages are Python and JavaScript. I also am skilled in HTML & CSS. I am a self-taught programmer and have been learning since 2019.

I started with Python in order to write some scripts to assist me with some menial tasks when I was working in customer service. I then learned the Django framework and built my first website slolearner.net (currently under reconstruction) a website to help people (like myself) who are trying to learn the Slovenian language.

Whilst using Django I improved my Vanilla JS, jQuery, HTML & CSS skills. Recently I have been using React with Django Rest API/AWS Amplify on the backend.

I have deployed all my past projects using pythonanywhere, but have been studying AWS and Docker for the past few months. I use Git version control and am comfortable with the Linux command line and using SSH.

I use this website to host and show some of my web apps. Check out my work for a detailed list of all my web apps.

Please contact me if you are looking to build a website or need help with hosting or deployment. I'll get back to you ASAP
